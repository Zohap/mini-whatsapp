# 📱 Mini WhatsApp Clone (Node.js + MongoDB + Express)

A simple WhatsApp-like chat application built using **Node.js, Express.js, MongoDB, Mongoose, EJS, and CSS**.  
This project demonstrates full CRUD operations with RESTful APIs and a clean WhatsApp-style UI.

---

## 🚀 Features

- View all chats
- Create new chat
- View single chat
- Edit message
- Delete chat
- Timestamp for messages
- Clean WhatsApp-style UI

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Method Override
- CSS

## ⚙️ Installation & Setup

### 1. Clone repository
git clone https://github.com/your-username/mini-whatsapp.git

### 2. Install dependencies
npm install

### 3. Start MongoDB
mongodb://127.0.0.1:27017/whatsappDB

### 4. Insert sample data (optional)
node init/init.js

### 5. Run project
node app.js
